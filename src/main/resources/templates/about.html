    About this project and the data that it represents

    This project was to gather info to describe and analyze desired skills in the tech job sector. It was a team project consisting of two Full Stack Engineers and four Data Analysts. The Full Stack team used Selenium to write programs to gather job skill data for three geographical areas: 1) Texas, 2) Georgia, and 3) Maryland/DC/Virginia. In addition, the Full Stack Engineers uploaded the data to a Postgres database for ease store and access. The Data Analysts then took this data and used Tableau to clean, explore, and analyze the data. The home page displays the visualizations. Please interact with the visualizations to see the analysis.
